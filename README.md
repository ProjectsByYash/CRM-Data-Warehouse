# CRM Analytical Data Warehouse

An end-to-end SQL data warehouse designed to transform raw, unstructured CRM data into highly optimized analytical schemas for business intelligence.

---

## 🚀 Project Overview

In modern sales operations, raw transactional data is rarely optimized for analytical queries. This project demonstrates data engineering methodologies by architecting a centralized data warehouse from raw CSV extracts. By applying dimensional modeling and advanced SQL techniques, this system reduces query latency and provides a single source of truth for sales performance metrics.

## 🛠️ Key Technical Features

* **Data Transformation (ETL/ELT):** Cleaned, structured, and staged raw sales team data into relational tables.
* **Dimensional Modeling:** Designed a Star Schema featuring robust Fact and Dimension tables
* **Performance Optimization:** Implemented indexing, efficient aggregation strategies, and optimized query execution paths for rapid reporting.
* **Actionable Insights:** Structured data to seamlessly power BI dashboards and executive reporting.
